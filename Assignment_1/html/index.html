Images of the Russian Empire: Colorizing the Prokudin-Gorskii Photo Collection



1- Problem Definition

The aim of this project is to automatically generate color images from digitized Prokudin-Gorskii glass plate images by applying some simple image processing techniques. For this, the red/green/blue color channels were removed from the digitized license plate images and the images were aligned to create an RGB color image with several visual imperfections.
